= Busy Thread

En este laboratorio veremos como encontrar cual de nuestros thread esta consumiendo el CPU.

1) Abrir una terminal y ejecutar

[source,bash]
----
./run.sh -l busy_thread
----

2) Listar los procesos Java

[source,bash]
----
jps -l
----

3) Ejecutar jvmtop

[source,bash]
----
./tools/jvmtop.sh -p <PID>
----
